Hacker School Match

A bookmarklet to compare Hacker Schooler skill sets.

Drag the link below to your bookmark bar and use the bookmarklet on the Hacker School private page. (Mozilla has more information on how to install and use bookmarklets, if you're unfamiliar.) A "find matches" link will appear underneath each person's photo. If there's no "find matches" link, that person has not listed any skills.
